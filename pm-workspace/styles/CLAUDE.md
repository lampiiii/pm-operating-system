# Styles

Writing style guides for different communication contexts.

## Contents

- `executive-email.md` — Executive email style guide: pyramid structure, lead with the ask, 3-5 bullet key points, clear next steps, pre-send checklist

## Usage

- Always apply the relevant style guide when drafting communications
- For executive emails: bottom-line-first, concise, action-oriented, respect their time
- When adding new style guides, create `audience-or-format.md` (e.g., `slack-updates.md`, `prd-writing.md`)

## Conventions

- One file per communication style/audience
- Each guide should include: core principles, structure/template, examples, and a checklist
