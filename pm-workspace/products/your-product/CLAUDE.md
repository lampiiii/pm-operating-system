# Your Product

This is a template product workspace. Rename this directory and customize for your product.

## Structure

- `strategy/` — Vision, mission, roadmap, OKRs
- `features/` — Active feature workspaces
  - `feature-template/` — Copy this folder for each new feature
- `product-ideas-backlogs/` — Product backlog tracking with interactive dashboard
- `research/` — User research, competitive analysis, market intelligence

## Workflow

1. Start new features by copying `features/feature-template/`
2. Track ideas in `product-ideas-backlogs/active-backlog.md`
3. Use the backlog dashboard at `/backlog` for visual management
4. Move completed features to an archive when done
